# Typebot – Fluxo de Agendamento

Este diretório contém o fluxo do chatbot desenvolvido no **Typebot**, responsável
pela coleta de dados do usuário para o agendamento automatizado de consultas.

## Fluxo Conversacional

1. Tela inicial de boas-vindas
2. Opção "Agendar consulta"
3. Coleta de dados:
   - Nome
   - Telefone
   - Motivo da consulta
   - Data desejada
   - Horário desejado
4. Envio dos dados via Webhook para o n8n
